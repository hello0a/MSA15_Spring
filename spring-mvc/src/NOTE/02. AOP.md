# Spring AOP
: 관점 지향 프로그래밍 (Aspect Oriented Programing)

## 관점
- 중심 관심사
 : 핵심 비즈니스 로직
- 횡단 관심사
 : 로깅, 보안, 트랜잭션 관리 등 공통 기능

## 주요 개념
- Aspect (관점)
 : 횡단 관심사를 모듈화한 것
- Advice (어드바이스)
 : 조인 포인트에서 실행되는 코드
- Join Point (조인 포인트)
 : 횡단 관심사가 적용될 수 있는 지점 (메서드 실 행 등)
    * @Before, @After, @Around, @AfterReturning, @AfterThrowing
- Pointcut (포인트컷)
 : 조인 포인트 및 타겟을 지정하는 표현식
